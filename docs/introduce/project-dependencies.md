安装依赖

```bash
$ yarn install
```

启动项目

```bash
$ yarn run start
```

组件库打包

```bash
$ yarn run build
```

文档库打包

```bash
$ yarn run docs:build
```

组件库打包tgz

```bash
$ yarn check
```
